Coding

  • Tạo một trang web tĩnh cho một thư viện ảnh bằng HTML và CSS
  • Bạn không được sử dụng JS hay bất kỳ một thư viện CSS nào
  • Trang web cần phải hiển thị đồng nhất với các cỡ màn hình khác nhau (responsive)
  • Thư viện ảnh gồm 3 cột
  • Các cột cần được đặt trong một container với độ rộng 900px và căn giữa trang web.
  • Xử lý sao cho ảnh ở hàng cuối có kích thước vừa với độ rộng của hàng.
